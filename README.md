# MultiClient

Client Application in .NET using WPF for reading OPC items with diferents protocols

<p align="center">
  <img src="Screen.png" alt="" width="350"/>
</p>

## Features
- Support for Modbus 
- Subscription to items changes.
- Save Items in JSON / XML

- EasyModbus (easymodbustcp.net)
